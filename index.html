Etch-a-Sketch Project

Project Brief:

  1. Create a page with 16x16 grid of square divs, using js
  2. Put these divs inside another 'container' div (which is directly in the html file)
  3. Use flexbox to make the divs appear as a grid (and not one per line) - careful with borders/margins as these adjust size of the squares
  4. Set up a hover effect so the grid divs change colour when mouse passes over them, leaving a pixelated trail through the grid like a pen would
  5. Add a button to top of screen that sends user a popup asking for number of squares per side for the new grid. Once entered, the default grid should be removed and replaced with a new grid, generated in the same total space as before (e.g. 960 px wide), so you have a new sketch pad. Limit the user input to max 100 squares.

Extra credit:

  1. Rather than colour change from black to white, each interaction should randomize the square's RGB value entirely
  2. Implement a progressive darkening effect where each interaction adds 10% more black or colour to the square. Objective is to achieve a completely black square only after 10 interactions.
